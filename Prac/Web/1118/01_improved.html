HyperText Markup Language

웹 페이지의 구조와 내용을 정의하는 마크업 언어


HyperText (하이퍼텍스트)

  1. 웹 페이지에서 다른 페이지로 연결되는 링크를 포함하는 텍스트
  2. 비선형적 정보 접근 가능
  3. 예: 위키피디아의 파란색 링크들

Markup (마크업)

  • 태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어
  • 표시(Mark)를 한다(Up)는 의미
  • 문서의 구조와 의미를 표현
  • 의미론적 강조
  • 단순 굵은 글씨

링크와 이미지

하이퍼링크 (Anchor)

네이버로 이동 (새 탭)
같은 페이지 내 이동 (테이블로)
실무 팁: 외부 링크는 target="_blank"로 새 탭에서 열고, 내부 링크는 같은 탭에서 여는 것이 일반적입니다.

이미지 (Image)

예시: 로고 이미지를 클릭하면 홈으로 이동
<a href="/"><img src="logo.png" alt="사이트 로고" width="150"></a>

폼 (Form) 요소

Label (레이블)

input 요소에 설명을 붙이는 태그. 클릭 시 연결된 input이 활성화되어 사용성 향상

두 가지 연결 방법:
1. for 속성 사용: <label for="username">이름</label> <input id="username">
2. 감싸기: <label><input> 이름</label>
